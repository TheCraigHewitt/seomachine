# Marketing Psychology & Mental Models

Read `.claude/product-marketing-context.md` if it exists.

## How to Use This Skill

1. Define the behavior to change.
2. Choose 2-4 relevant models.
3. Apply models to specific copy, UX, offer, or channel decisions.
4. Use ethical constraints: clarity, truthful claims, real scarcity only.

## Foundational Thinking Models

| Model | Marketing Application |
|-------|------------------------|
| First Principles | Break strategy into core assumptions before choosing channels. |
| Jobs to Be Done | Frame offers by outcome, not feature list. |
| Circle of Competence | Focus on channels where you can execute well. |
| Inversion | List failure modes first, then design against them. |
| Occam's Razor | Check simple causes before complex theories. |
| Pareto Principle | Put most effort into the few activities driving most results. |
| Local vs. Global Optima | Avoid optimizing tactics inside a weak strategy. |
| Theory of Constraints | Find and fix the single funnel bottleneck first. |
| Opportunity Cost | Compare each tactic against the best alternative use of time/budget. |
| Diminishing Returns | Stop over-investing once incremental gains flatten. |
| Second-Order Thinking | Evaluate downstream effects (e.g., discounting and brand erosion). |
| Map ≠ Territory | Validate models with real user feedback, not dashboards alone. |
| Probabilistic Thinking | Plan scenarios with likelihoods instead of certainty. |
| Barbell Strategy | Put most budget in proven plays, a minority in experiments. |

## Understanding Buyers & Human Psychology

| Model | Marketing Application |
|-------|------------------------|
| Fundamental Attribution Error | Diagnose process friction before blaming "low intent" users. |
| Mere Exposure Effect | Maintain consistent repeated messaging across touchpoints. |
| Availability Heuristic | Use vivid examples and recent proof to reduce uncertainty. |
| Confirmation Bias | Start from beliefs users already hold, then expand. |
| Lindy Effect | Prefer durable principles over tactic fads. |
| Mimetic Desire | Show credible peers using the product. |
| Sunk Cost Fallacy | Kill weak campaigns despite past spend. |
| Endowment Effect | Use free trials/demos to create ownership. |
| IKEA Effect | Add lightweight setup/customization to increase commitment. |
| Zero-Price Effect | Use free strategically for entry or activation moments. |
| Present Bias | Lead with immediate value, then long-term gains. |
| Status-Quo Bias | Reduce switching friction and perceived risk. |
| Default Effect | Set ethical defaults for the preferred action. |
| Paradox of Choice | Reduce options when conversion stalls. |
| Goal-Gradient Effect | Show progress to increase completion. |
| Peak-End Rule | Design memorable peaks and strong endings. |
| Zeigarnik Effect | Use incomplete-state prompts to bring users back. |
| Pratfall Effect | Admit bounded weaknesses to increase trust. |
| Curse of Knowledge | Remove jargon and test copy with new users. |
| Mental Accounting | Reframe price in customer-relevant units. |
| Regret Aversion | Reduce downside fear with guarantees and trials. |
| Bandwagon/Social Proof | Show adoption signals (logos, testimonials, counts). |

## Influencing Behavior & Persuasion

| Model | Marketing Application |
|-------|------------------------|
| Reciprocity | Give useful value before asking for conversion. |
| Commitment & Consistency | Ask for small commitments before larger ones. |
| Authority Bias | Add credible expertise and proof of competence. |
| Liking / Similarity | Use relatable voice and audience-matched examples. |
| Unity Principle | Build shared identity with your audience segment. |
| Scarcity / Urgency | Use only genuine limits and deadlines. |
| Foot-in-the-Door | Sequence asks from low to high commitment. |
| Door-in-the-Face | Use contrast carefully to make core offer clearer. |
| Loss Aversion | Frame cost of inaction, not just upside. |
| Anchoring | Set price context with higher anchor first. |
| Decoy Effect | Add a dominated option to clarify target choice. |
| Framing Effect | Present equivalent facts in the most decision-useful frame. |
| Contrast Effect | Make before/after difference explicit. |

## Pricing Psychology

| Model | Marketing Application |
|-------|------------------------|
| Charm Pricing | Use .99/.95 when emphasizing value perception. |
| Rounded-Price Fluency | Use round numbers for premium positioning. |
| Rule of 100 | Under $100 use % discount; above $100 use absolute discount. |
| Price Relativity | Use good-better-best to guide selection. |
| Mental Accounting (Pricing) | Reframe monthly price into daily/workflow value. |

## Design & Delivery Models

| Model | Marketing Application |
|-------|------------------------|
| Hick's Law | Reduce choices and fields to decrease drop-off. |
| AIDA | Sequence content: attention, interest, desire, action. |
| Rule of 7 | Plan multi-touch journeys, not single-touch expectations. |
| Nudge Theory | Improve choice architecture without removing agency. |
| BJ Fogg Model | Ensure motivation, ability, and prompt coexist. |
| EAST Framework | Make actions easy, attractive, social, timely. |
| COM-B | Check capability, opportunity, motivation gaps. |
| Activation Energy | Shrink first-step effort to increase starts. |
| North Star Metric | Align campaigns to one value-delivery metric. |
| Cobra Effect | Validate incentives to avoid gaming/backfires. |

## Growth & Scaling Models

| Model | Marketing Application |
|-------|------------------------|
| Feedback Loops | Build systems where usage creates more acquisition. |
| Compounding | Prioritize repeatable actions with long-term carryover. |
| Network Effects | Highlight value that increases with each additional user. |
| Flywheel Effect | Connect acquisition, activation, retention loops. |
| Switching Costs | Build ethical stickiness via integrations and workflows. |
| Exploration vs. Exploitation | Split budget between optimization and discovery. |
| Critical Mass / Tipping Point | Concentrate distribution where momentum can compound. |
| Survivorship Bias | Learn from failed campaigns, not only visible wins. |

## Quick Reference

| Challenge | Useful Models |
|-----------|----------------|
| Low conversions | Hick's Law, Activation Energy, Fogg, Paradox of Choice |
| Price objections | Anchoring, Framing, Mental Accounting, Loss Aversion |
| Weak trust | Authority, Social Proof, Reciprocity, Pratfall |
| Slow activation | Goal-Gradient, Default Effect, Commitment |
| Rising churn | Endowment, Switching Costs, Status-Quo Bias |
| Stalled growth | Theory of Constraints, Compounding, Feedback Loops |
