# Ad Copy Templates Reference

## Primary Text Formulas

### Problem-Agitate-Solve (PAS)

```text
[Problem]
[Agitate the consequence]
[Introduce solution]
[CTA]
```

**Example**
> Spending hours on manual reporting?
> While you are in spreadsheets, decisions stall.
> [Product] automates reporting in minutes.
> Start your free trial ->

### Before-After-Bridge (BAB)

```text
[Current state]
[Desired state]
[Bridge: your product]
```

**Example**
> Before: approvals scattered across tools.
> After: one tracked workflow with owners and deadlines.
> [Product] connects your stack and automates approvals.

### Social Proof Lead

```text
[Specific proof]
[What you do]
[CTA]
```

**Example**
> "We cut reporting time by 75%."
> [Product] automates repetitive reporting workflows.
> See how it works ->

### Feature-Benefit Bridge

```text
[Feature]
So that [operational benefit]
Which means [business impact]
```

### Direct Response

```text
[Outcome claim]
[Proof]
[CTA + urgency only if true]
```

## Headline Formulas

### Search Ads

| Formula | Example |
|---------|---------|
| [Keyword] + [Benefit] | "Project Management That Teams Use" |
| [Action] + [Outcome] | "Automate Reports | Save 10 Hours Weekly" |
| [Question] | "Tired of Manual Data Entry?" |
| [Number] + [Benefit] | "500+ Teams Trust [Product]" |
| [Offer] + [Keyword] | "Free Project Management | No Card" |

### Social Ads

| Type | Example |
|------|---------|
| Outcome hook | "How we tripled conversion rate" |
| Curiosity hook | "The reporting hack most teams miss" |
| Contrarian hook | "Why we stopped using [common tool]" |
| Specificity hook | "The exact template we use for ..." |
| Question hook | "What if reporting took 10 minutes?" |

## CTA Variations

### Soft CTA (awareness)
- Learn More
- See How It Works
- Get the Guide
- Read the Case Study

### Hard CTA (conversion)
- Start Free Trial
- Book a Demo
- Get Started Free
- Claim Your Discount

### Urgency CTA (only when true)
- Offer Ends [Date]
- Last Chance
- Limited Spots Left

### Action-Outcome CTA
- Start Saving Time Today
- Get Your Free Report
- Calculate Your ROI

## Platform Copy Constraints

### Google Search Ads
- Headlines: up to 30 chars each
- Descriptions: up to 90 chars each
- Use keyword + differentiator + CTA variants

### Meta Ads
- Primary text: first 125 chars matter most
- Headline: keep concise
- Hook in first line; test short and medium copy

### LinkedIn Ads
- Intro text: up to 600 chars (shorter often clearer)
- Headline: up to 200 chars
- Use specific professional outcomes and proof

## Copy Testing Priority

1. Hook/angle
2. Headline
3. Core benefit
4. CTA
5. Proof elements
