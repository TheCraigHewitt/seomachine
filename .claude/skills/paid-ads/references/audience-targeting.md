# Audience Targeting Reference

## Google Ads Audiences

### Search Campaign Targeting
- Use match types intentionally: exact, phrase, broad.
- Add audiences in observation first, then tighten targeting from data.
- Use RLSA to bid higher for prior visitors searching high-intent terms.

### Display/YouTube Targeting
- Custom intent from converting keyword themes
- In-market for active category research
- Affinity for broader awareness
- Customer match for first-party audiences

## Meta Audiences

### Core Audiences
- Interest, demographic, and behavior targeting
- Start broad enough for delivery, then refine from performance

### Custom Audiences
- Website visitors (by page/time window)
- Customer lists
- Video/view engagement audiences
- App activity audiences

### Lookalike Audiences
- Best source: high-LTV or recent purchasers
- Typical sizing:
  - 1%: highest similarity
  - 1-3%: balance
  - 3-5%: scale
  - 5-10%: broad awareness

## LinkedIn Audiences

### Job-Based Targeting
- Job title + seniority + function is core stack
- Skills can expand reach but are less precise alone

### Company-Based Targeting
- Company size + industry + named account lists (ABM)
- Growth/hiring signals can improve timing

### High-Performing Combinations

| Use Case | Targeting Combination |
|----------|----------------------|
| Enterprise sales | Company size 1000+ + VP/CXO + industry |
| SMB sales | Company size 11-200 + manager/director + function |
| Developer tools | Skills + function + company type |
| ABM | Named accounts + decision-maker titles |

## Twitter/X Audiences
- Follower lookalikes
- Keyword/topic targeting
- Tailored audiences from site/customer lists

Best use: awareness and conversation capture with strong creative iteration.

## TikTok Audiences
- Demographic + interest + behavior + custom audiences
- Creative quality typically matters more than narrow targeting

## Audience Size Guidelines

| Platform | Typical Minimum | Typical Operating Range |
|----------|------------------|-------------------------|
| Google Search | 1,000 monthly searches | 5,000-50,000 |
| Meta | 100,000 | 500K-10M |
| LinkedIn | 50,000 | 100K-500K |
| Twitter/X | 50,000 | 100K-1M |
| TikTok | 100,000 | 1M+ |

Too narrow: limited learning + high costs.  
Too broad: weak relevance and spend efficiency.

## Exclusion Strategy

Always consider excluding:
- Existing customers from prospecting
- Recent converters
- Low-intent bounces
- Internal employees
- Irrelevant page visitors (careers/support)
