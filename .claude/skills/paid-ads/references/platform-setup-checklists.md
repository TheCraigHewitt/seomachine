# Platform Setup Checklists

## Google Ads Setup

### Account Foundation
- [ ] Account, billing, timezone, currency configured
- [ ] Team access configured

### Conversion Tracking
- [ ] Google tag installed sitewide
- [ ] Conversions created (lead/signup/purchase)
- [ ] Values assigned where relevant
- [ ] Test conversion verified

### Analytics and Audiences
- [ ] GA4 linked, auto-tagging enabled
- [ ] Remarketing audiences created
- [ ] Customer lists uploaded (if available)

### Campaign Readiness
- [ ] Negative keyword lists prepared
- [ ] Location/language/device settings reviewed
- [ ] Ad extensions added
- [ ] Brand campaign coverage confirmed

## Meta Ads Setup

### Business Foundation
- [ ] Business Manager + ad account + billing set
- [ ] Team roles configured

### Pixel and Events
- [ ] Meta Pixel installed
- [ ] Standard events configured (PageView, ViewContent, Lead, Purchase)
- [ ] Conversions API configured (if possible)
- [ ] Test events validated

### Domain and Audiences
- [ ] Domain verified
- [ ] Aggregated events prioritized
- [ ] Custom audiences and lookalikes created

### Creative and Compliance
- [ ] Required creative sizes ready
- [ ] UTM tags on all destination URLs
- [ ] Policy compliance check complete

## LinkedIn Ads Setup

### Campaign Manager Foundation
- [ ] Account, Company Page link, billing, access configured

### Insight Tag and Conversions
- [ ] Insight Tag installed and verified
- [ ] Conversion events configured

### Audiences and Forms
- [ ] Website/contact/company matched audiences set
- [ ] Lead gen forms configured (if used)
- [ ] Form privacy policy and thank-you flow configured

### Creative and Budget
- [ ] Creative meets format requirements
- [ ] Intro/headline copy within limits
- [ ] Budget matches expected CPC reality

## Twitter/X Ads Setup

### Foundation and Tracking
- [ ] Ads account and billing configured
- [ ] Pixel and conversion events verified

### Audience and Creative
- [ ] Tailored audiences uploaded
- [ ] Follower lookalikes/keywords selected
- [ ] Creative and card formats validated

## TikTok Ads Setup

### Foundation and Tracking
- [ ] Ads Manager + business verification + billing complete
- [ ] Pixel/events configured and tested

### Audience and Creative
- [ ] Custom/lookalike audiences created
- [ ] Vertical native-style videos ready
- [ ] Strong first 3-second hooks present

## Universal Pre-Launch Checklist

- [ ] Real conversion test passed
- [ ] Landing page speed and mobile UX checked
- [ ] UTM schema validated
- [ ] Dates/budgets/frequency caps reviewed
- [ ] Targeting and exclusions reviewed
- [ ] Reporting dashboard prepared
