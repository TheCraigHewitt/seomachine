# Pricing Strategy

Read `.claude/product-marketing-context.md` if it exists.

## Before Starting

Capture:
1. Business model and GTM motion.
2. Target segments and alternatives.
3. Current monetization performance.
4. Pricing objective (growth, margin, expansion, market position).

## Pricing Fundamentals

### Three Pricing Axes

| Axis | Question |
|------|----------|
| Packaging | What is included in each tier? |
| Value metric | What do you charge for? |
| Price point | How much does each option cost? |

### Value-Based Rule
Set price between:
- next best alternative (floor)
- customer-perceived value (ceiling)

## Value Metrics

| Metric | Works Best For | Example |
|--------|----------------|---------|
| Per seat | Collaboration workflows | Slack-like tools |
| Per usage | Consumption products | API/data platforms |
| Per contact/record | CRM/marketing data | Email/CRM tools |
| Per transaction | Payments/marketplaces | Payments software |
| Flat fee | Simple fixed-scope product | Team productivity apps |

Decision rule: if value increases when the metric increases, metric fit is strong.

Use detailed tier and persona packaging reference: [references/tier-structure.md](references/tier-structure.md)

- **Better**: recommended default
- **Best**: premium/enterprise capture

## Pricing Research

Use methods from [references/research-methods.md](references/research-methods.md):
- Van Westendorp (price sensitivity range)
- MaxDiff (feature value ranking)
- WTP/conjoint methods
- Usage-value correlation analysis

## When to Raise Prices

### Signals
- Typical: very high win/conversion rates can indicate underpricing.
- Typical: strong retention with expanding usage can support increase testing.
- Product value materially improved since last price update.

### Increase Strategies
1. New-customer increase first.
2. Advance notice for existing customers.
3. Increase with added value.
4. Repackage tiers instead of only changing numbers.

## Pricing Page Best Practices

### Core Elements
- Clear comparison table
- Recommended tier highlight
- Monthly/annual toggle
- Tier-specific CTA
- FAQ and trust proof

### Psychology
- Anchoring (show higher option clearly)
- Decoy design (middle tier value clarity)
- Charm vs rounded pricing by positioning

## Pricing Checklist

### Before Setting Prices
- [ ] Persona segments defined
- [ ] Competitor and alternative pricing mapped
- [ ] Value metric selected
- [ ] WTP research completed
- [ ] Feature-to-tier mapping drafted

### Launch Readiness
- [ ] Tier count finalized
- [ ] Tier differentiation clear
- [ ] Annual policy and discount set
- [ ] Enterprise/custom path defined
- [ ] Change communication plan prepared
