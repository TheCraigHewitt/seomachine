# SEO Guidelines for Castos Content

Use this as the default SEO standard for Castos blog production and updates.

## Content Length Targets
- **Standard blog post**: 1,500-3,000 words (target 2,000-2,500)
- **Pillar/guide**: up to 5,000 words
- **How-to**: 1,500-2,500 words
- **News/update**: 800-1,200 words

Rules:
- Prefer concise depth over padding.
- Split overly broad topics into a series.

## Keyword Optimization

### Before Writing
1. Select primary keyword.
2. Check search volume and difficulty.
3. Review top competitors.
4. Select 3-5 secondary keywords.
5. Note LSI/semantic terms.

### Density Targets
- **Primary keyword**: 1-2%
- **Secondary keywords**: 0.5-1% each

### Required Placement
Primary keyword should appear in:
- [ ] H1
- [ ] First 100 words
- [ ] 2-3 H2s
- [ ] Conclusion
- [ ] Meta title
- [ ] Meta description
- [ ] URL slug

### Integration Rules
- Write naturally; avoid stuffing.
- Use variations, not exact repetition.
- Match intent and conversational phrasing.

## Content Structure

### Heading Rules
- One H1 only, keyword included naturally.
- 4-7 H2s for standard posts.
- Proper H2 -> H3 nesting (no skipped levels).

### Article Template
```markdown
# [H1 with primary keyword]

## Introduction (150-250 words)
- Hook
- Problem
- Promise
- Keyword in first 100 words

## [H2 Section 1]
## [H2 Section 2]
## [H2 Section 3]
## [H2 Section 4]

## Conclusion (150-250 words)
- Key takeaways
- Keyword reinforcement
- Clear CTA
```

## Meta Elements

### Meta Title
- Length: 50-60 chars
- Include primary keyword
- Compelling and unique
- Accurate to page content

### Meta Description
- Length: 150-160 chars
- Include keyword naturally
- Value proposition + CTA
- Complete sentence

### URL Slug
- Lowercase, hyphenated, concise
- Include primary keyword
- Prefer 3-5 words

## Internal Linking
- **Minimum**: 3 links
- **Optimal**: 4-5 links
- **Maximum**: 7 links (unless very long article)

Include:
- 1-2 pillar links
- 2-3 related blog links
- 0-1 product link (only if natural)
- 0-1 resource link

Anchor text:
- Descriptive and contextual
- Avoid generic phrases like "click here"
- Check targets in `@context/internal-links-map.md`

## External Linking
- **Minimum**: 2 authority links
- **Optimal**: 3-4 links

Use for:
- statistics, studies, claims, and cited guidance

Quality bar:
- relevant
- credible
- recent (especially data)
- functional (no broken links)

## Readability Targets
- Reading level: 8th-10th grade
- Avg sentence length: 15-20 words
- Paragraphs: 2-4 sentences
- Prefer active voice
- Add subheadings every ~300-400 words

## Content Quality Standards
- Accurate, current, and original
- Back key claims with sources
- Useful and actionable for podcast creators
- Aligned with E-E-A-T principles

## Image SEO
- Descriptive file names
- Relevant alt text (<=125 chars)
- Optimize size for load speed
- Place images where they support nearby text

## Featured Snippet Optimization
Use when relevant:
- **Question snippet**: H2 question + 40-60 word direct answer
- **List snippet**: concise numbered/bulleted steps
- **Table snippet**: clear comparison table
- **Definition snippet**: short, direct definition first

## Content Refresh Triggers
Update when:
- article is 12+ months old
- rankings decline
- stats/processes become outdated
- competitor content overtakes quality/completeness

Refresh:
- data
- examples/screenshots
- internal links
- meta elements and keyword focus if needed

## SEO Checklist
### Content
- [ ] Appropriate word count for type
- [ ] Primary + secondary keywords mapped
- [ ] Unique value vs competitors
- [ ] Facts verified

### Structure
- [ ] One H1 with keyword
- [ ] 4-7 H2s
- [ ] 2-3 H2s with keyword variations
- [ ] Keyword in intro and conclusion

### Meta
- [ ] Meta title 50-60 chars
- [ ] Meta description 150-160 chars
- [ ] URL slug optimized

### Links
- [ ] 3-5 internal links
- [ ] 2-3+ external authority links
- [ ] No broken links

### Readability
- [ ] 8th-10th grade level
- [ ] Sentence/paragraph targets met
- [ ] Scannable formatting

### Quality
- [ ] Brand voice aligned
- [ ] Sources cited
- [ ] Actionable guidance present
- [ ] Clear CTA

## Tools
- Keyword research: Ahrefs, SEMrush, Google Keyword Planner
- Technical/rank checks: Google Search Console, Ahrefs, SEMrush
- Readability: Hemingway, Grammarly

Remember: prioritize reader value first; SEO should reinforce usefulness, not replace it.
