## Word Choice & Usage

### Preferred Terms
**Say This -> Not That**
- podcast creators -> podcasters
- podcast hosting -> podcast storage
- listeners -> subscribers
- analytics -> stats
- monetize -> monetise
- [Add Castos-specific mappings]

### Avoid
- unnecessary filler (very, really, actually)
- unclear jargon without explanation
- passive voice when active is clearer
- generic link text ("click here")

### Inclusive Language
- gender-neutral phrasing
- globally understandable language
- accessible descriptions

## Formatting Standards

### Emphasis
- **Bold**: key concepts/takeaways
- *Italics*: light emphasis or titles
- Underline: avoid except links
- ALL CAPS: acronyms only

### Lists
- Bullets for unordered items
- Numbered lists for sequences/priorities
- Use consistent punctuation
- Keep list structure parallel

### Links
- Use descriptive anchor text
- Keyword-rich when natural
- Open external links in new tab: [Yes/No]

### Code & Technical Elements
- Inline code in backticks for URLs, file names, commands
- Fenced code blocks for multi-line examples

## Content Structure

### Introduction (150-250 words)
1. Hook
2. Problem
3. Promise
4. Credibility (optional)

Keyword in first 100 words.

### Section Length
- Minimum: ~150 words
- Ideal: 250-350 words
- Maximum: ~500 words before splitting

### Conclusion (150-250 words)
1. Recap
2. Action step
3. CTA
4. Optional forward-looking close

## SEO Style Standards
- Meta title: 50-60 chars
- Meta description: 150-160 chars
- URL slug: lowercase, hyphenated, 3-5 words
- Alt text: descriptive, <=125 chars, no "image of"
