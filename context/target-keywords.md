# Target Keywords & Topic Clusters

Organize target terms into clusters to guide content planning, internal linking, and ranking growth.

## Cluster Template
Each cluster should include:
- **Pillar Keyword**: primary high-value term
- **Cluster Keywords**: 5-10 related subtopics
- **Long-Tail Keywords**: 10-15 specific phrases
- **Search Intent**: informational/commercial/transactional

## Usage Guidelines
### When Writing New Content
1. Select the right cluster and target keyword.
2. Link to pillar and sibling cluster content.
3. Add semantic terms naturally.

### When Optimizing Existing Content
1. Reconfirm intent match.
2. Check for cannibalization.
3. Update critical keyword placements.

### When Planning Calendar
1. Prioritize high-value, achievable keywords.
2. Balance pillar and cluster content.
3. Track changes monthly and refine quarterly.
