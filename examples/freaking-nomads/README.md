# Freaking Nomads — Example Files

This directory contains fully completed example context files for Freaking Nomads. These serve as reference for anyone adapting this SEO Machine workspace for a new project — showing what a fully configured workspace looks like.

For active content generation, reference the `context/` directory instead. These files are mirrors of the context files, kept here as examples.

## Files

- `brand-voice.md` — FN brand voice, tone rules, banned words, author personas
- `writing-examples.md` — Real verbatim excerpts from freakingnomads.com by content type
- `features.md` — FN content offerings and community
- `internal-links-map.md` — All key FN URLs organized by category
